# ⏳ Chronos

**Chronos** — это легкий и интуитивно понятный CLI тайм-трекер.

## ✨ Основные возможности

- 🚀 **Быстрый старт**: Запуск и остановка задач одной командой.
- ➕ **Гибкое добавление**: Добавляй задачи постфактум по длительности или времени начала.
- 📈 **Визуальный Таймлайн**: Потрясающая ASCII-визуализация твоего дня.
- 📉 **Компактный режим**: Группировка повторяющихся задач и интервалов.
- 🇷🇺 **Локализация**: Привычный формат дат (ДД.ММ.ГГГГ) и времени.
- 🧹 **Управление**: Удобный просмотр и удаление записей по ID.

## 🛠 Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/egorshuk/chronos.git
   cd chronos
   ```

2. Установите проект в режиме разработки:
   ```bash
   pip install -e .
   ```

## 🚀 Использование

### Трекинг в реальном времени
```bash
chronos start "Разработка API"
# ... работаем ...
chronos stop
```

### Добавление прошедших задач
```bash
# Добавить 1.5 часа работы, которая закончилась сейчас
chronos add "Учеба" 1:30

# Добавить задачу, указав время старта
chronos add "Митинг" 0:30 --start 14:00
```

### Просмотр и Аналитика
```bash
# Список всех задач за сегодня
chronos show --today

# Визуальный таймлайн дня
chronos timeline

# Компактный отчет с группировкой
chronos timeline --compact
```

### Удаление
```bash
chronos delete 5
```
